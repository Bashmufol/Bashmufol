## 🚀 Featured Projects

### ⭐ [Authguard](https://motion-markets-factor-prostores.trycloudflare.com/) — Flagship

Multi-tenant B2B access control for teams that need consistent authorization across multiple applications without rebuilding RBAC in every service. Supports project-scoped roles and permissions, API key governance, JWT token exchange, centralized authorize checks, MFA step-up, and an append-only audit trail.

`Java 21` · `Spring Boot 4` · `Microservices` · `PostgreSQL` · `Liquibase` · `Redis` · `RabbitMQ` · `JWT/JWKS` · `OAuth2` · `Docker` · `Prometheus`

🔗 [Live demo](https://motion-markets-factor-prostores.trycloudflare.com/) · [Backend system design](https://bashmufol.vercel.app/projects/authguard/architecture)

---

### [EvoCharge](https://github.com/Bashmufol/EvoCharge)

EV charging intelligence for Nigeria's fragmented public charging market. Unifies operator data, ranks stations with an EvoScore engine, streams live status over SSE, and offers a natural-language charging advisor backed by Amazon Bedrock.

`Java 21` · `Spring Boot` · `DynamoDB` · `AWS` · `AWS CDK` · `SSE` · `Docker` · `CI/CD`

🔗 [Live demo](https://d2zlb8ixz2ivq7.cloudfront.net/) · [Source code](https://github.com/Bashmufol/EvoCharge) · [Backend system design](https://bashmufol.vercel.app/projects/evocharge/architecture)

---

### [Legally](https://github.com/Bashmufol/Legally)

Legal information platform with multimodal consults (text, voice, files), jurisdiction resolution, and ordered multi-LLM failover. Session-scoped history on Cloud SQL, Firebase auth and storage, deployed on Cloud Run.

`Java 21` · `Spring Boot 4` · `Spring AI` · `PostgreSQL` · `Firebase` · `GCP` · `Docker`

🔗 [Live demo](https://legally-7f34d.web.app/) · [Source code](https://github.com/Bashmufol/Legally) · [Backend system design](https://bashmufol.vercel.app/projects/legally/architecture)

---

### [Terratrade](https://terratrade-50d.pages.dev/)

Cross-border agricultural trade platform for farmers, aggregators, and international buyers. Covers harvest listings, geo-based sourcing, export batching, escrow-backed payments, compliance scorecards, and RFQ matching.

`Java 21` · `Spring Boot 4` · `PostgreSQL` · `PostGIS` · `Flyway` · `JWT` · `Docker` · `OpenAPI`

🔗 [Live demo](https://terratrade-50d.pages.dev/) · [Backend system design](https://bashmufol.vercel.app/projects/terratrade/architecture)
